# `upperCaseFirst`
To capitalise the first letter in the string provided. _Defined on line [344](../../F.js#344)_.

## Use
```
upperCaseFirst(word);
```

### Parameters
* `word` (_string_): The string to process.

### Returns
* `String`
* `Error`

## Example
```javascript
Expose('upperCaseFirst');

upperCaseFirst('sharikul'); // returns "Sharikul"
```
