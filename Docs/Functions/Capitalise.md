# `Capitalise`
To capitalise each word in the string provided. _Defined on line [363](../../F.js#L363)_.

## Use
```
Capitalise(string);
```

### Parameters
* `string` (_string_): The string to process.

### Returns
* `String`
* `Error`

## Example
```javascript
Expose('Capitalise');

Capitalise('my name is sharikul'); // returns "My Name Is Sharikul"
```
