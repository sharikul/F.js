# `inArray`
To check the existence of a value in an array. _Defined on line [396](../../F.js#L396)_.

## Use
```
inArray(array, value);
```

### Parameters
* `array` (_array_): The array to check.
* `value` (_string_): The value to search for.

### Returns
* `Boolean`
* `TypeError`

## Example
```javascript
Expose('inArray');

inArray(['Sharikul', 18], 'Sharikul'); // true
```
