# `lTrim`
To remove whitespace on the left of a string. _Defined on line [438](../../F.js#L438)_.

## Use
```
lTrim(word);
```

### Parameters
* `word` (_string_): The string to process.

### Returns
* `String`
* `Error`

## Example
```javascript
Expose('lTrim');

lTrim('    Sharikul     '); // returns "Sharikul     "
```
