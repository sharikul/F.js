# `rTrim`
To remove whitespace from the right-side of a string. _Defined on line [453](../../F.js#L453)_.

## Use
```
rTrim(word);
```

### Parameters
* `word` (_string_): The string to process.

### Returns
* `String`
* `Error`

## Example
```javascript
Expose('rTrim');

rTrim('     Sharikul     '); // returns "     Sharikul"
```
