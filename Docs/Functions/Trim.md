# `Trim`
To remove whitespace from around the string provided. _Defined on line [423](../../F.js#L423)_.

## Use
```
Trim(word);
```

### Parameters
* `word` (_string_): The string to process.

### Returns
* `String`
* `TypeError`

## Example
```javascript
Expose('Trim');

Trim('    Sharikul    '); // returns "Sharikul"
```
