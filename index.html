    F.js - a functional JavaScript library

    F.js is a library of functions which do various tasks and are designed to aid your development whenever required. With the exception of the Expose function, you are required to load other functions whenever required. Individual function names start with a capital letter whilst multiple-worded functions are camel-cased.

    Meet the functions

    Expose

    The Expose function is the only F.js function that is available from the window scope by default. Its purpose is to load other F.js functions into the scope specified (window by default) for your use.

    Implementation:

    window.Expose = function(fn, scope);

    Source:

    https://github.com/sharikul/F.js/blob/master/F.js#L754

    Returns:

    Boolean

    Predefined values:

    '*' (caret symbol)

    Usage:

    // Our aim is to load the toArray, forEach, and Concat functions into the window scope
Expose(['toArray', 'forEach', 'Concat']);
// Done!

// You can also load individual functions by supplying a string
Expose('toArray');

// And you can expose a function into another scope, e.g. a custom object
var fns = {}
Expose(['toArray', 'forEach', 'Concat'], fns);

// You can choose to load every F.js function into the scope specified
Expose('*');

    toArray

    The toArray function can be used to convert array like objects, and hash tables into arrays.

    Implementation:

    toArray: function(nonArray);

    Source:

    https://github.com/sharikul/F.js/blob/master/F.js#L21

    Returns:

    Array

    Predefined values:

    None

    Usage:

    // This will return a nodelist.
var divs = document.querySelectorAll('div');

// Though it appears as an array, it isn't treated as one. toArray to the rescue!
Expose('toArray');
var div_s = toArray(divs);
// Done - div_s contains the elements from divs, and can be used as an array

    forEach

    You can use forEach to loop over arrays and objects.

    Implementation:

    forEach: function(object, callback);

    Source:

    https://github.com/sharikul/F.js/blob/master/F.js#L43

    Returns:

    Mixed

    Predefined values:

    None

    Usage:

    Expose('forEach');

var array = [1, 2, 3, 4],
    obj = {name: 'Sharikul', age: 18};

forEach(array, function(num) {
 console.log(num); // Logs 1, 2, 3, 4
});

forEach(obj, function(value, key) {
 console.log('My ' + key + ' is ' + value); // Logs 'My name/age is Sharikul/18'
});

    Range

    Range can be used to generate a sequence of numbers.

    Implementation:

    Range: function(first, second);

    Source:

    https://github.com/sharikul/F.js/blob/master/F.js#L67

    Returns:

    Array

    Predefined values:

    None

    Usage:

    Expose('Range');

// First, let's generate a range low-to-high.
console.log( Range(1, 5) ); // Logs [1, 2, 3, 4, 5]

// Now the opposite!
console.log( Range(5, 1) ); // Logs [5, 4, 3, 2, 1]

    Format

    You can use Format to create formatted strings using numerical indexes to denote where a specific value should be placed. Values should be supplied in order after the string.

    Implementation:

    Format: function(string);

    Source:

    https://github.com/sharikul/F.js/blob/master/F.js#L103

    Returns:

    String

    Predefined values:

    None

    Usage:

    Expose('Format');

console.log( Format('Hello, {0}! You are {1} years old!', 'Sharikul', 18) ); // Logs 'Hello, Sharikul! You are 18 years old!'

    Concat

    You can use Concat to merge several different arrays into one master array. Each argument you supply must be an array and Concat doesn't merge sub-arrays.

    Implementation:

    Concat: function();

    Source:

    https://github.com/sharikul/F.js/blob/master/F.js#L149

    Returns:

    Array

    Predefined values:

    None

    Usage:

    Expose('Concat');

var array1 = [1,2,3,4],
    array2 = ['name', 'Sharikul'],
    array3 = ['location', 'London'];

console.log( Concat(array1, array2, array3) ); // Logs [1, 2, 3, 4, 'name', 'Sharikul', 'location', 'London']

    attachEvent

    attachEvent provides you with an interface to 'attach' event handlers to various elements. It enables you to attach one event, and its callback to a variety of elements as well.

    Implementation:

    attachEvent: function(event, callback, element, bubble);

    Source:

    https://github.com/sharikul/F.js/blob/master/F.js#L182

    Returns:

    Boolean

    Predefined values:

    None

    Usage:

    Expose('attachEvent');

    <body>
 <button id="button">Click me, an alert will appear!</button>
 <p id="paragraph">No click me, an alert will appear!</button>
</body>

    attachEvent('click', function() {
 alert('You have clicked!');
}, ['#button', '#paragraph']); 

// OR
attachEvent('click', function() {
 alert('You have clicked!');
}, '#button');

    typeOf

    The typeOf function works exactly like typeof, except that it has added support to recognize arrays, nodelists, and so on.

    Implementation:

    typeOf: function(element);

    Source:

    https://github.com/sharikul/F.js/blob/master/F.js#L222

    Returns:

    String

    Predefined values:

    None

    Usage:

    Expose('typeOf');

console.log( typeOf([1,2]) ); // Logs 'array'
console.log( typeOf({}) ); // Logs 'object'
console.log( typeOf( document.querySelectorAll('div') ) ); // Logs 'nodelist'

    Store

    The Store function acts as a wrapper around the HTML 5 specific localStorage and sessionStorage functions.

    Implementation:

    Store: function(access, key, value);

    Source:

    https://github.com/sharikul/F.js/blob/master/F.js#L238

    Returns:

    Mixed

    Predefined values:

    None

    Usage:

    Expose('Store');

// Set a value in the session storage. Works the same way for local storage.
Store('session.set', 'name', 'Sharikul');

console.log( Store('session.get', 'name') ); // Logs 'Sharikul'

    addStyle

    You can use addStyle to add CSS styling to the specified element(s). Since this function internally uses getElement, you can supply multiple elements in an array, or just supply a string. You should supply styles in an object.

    Implementation:

    addStyle: function(element, style);

    Source:

    https://github.com/sharikul/F.js/blob/master/F.js#L433

    Returns:

    Array

    Predefined values:

    None

    Usage:

    Expose('addStyle');

    <h1 id="title">Title</h1>
<h3 id="subtitle">Subtitle</h1>

    addStyle(['#title', '#subtitle'], {
 fontFamily: 'sans-serif',
 fontWeight: 'bold'
})

// OR
addStyle('#title', {
 fontFamily: 'sans-serif',
 fontWeight: 'bold'
});

    <h1 id="title" style="font-family: sans-serif; font-weight: bold;">Title</h1>
<h3 id="subtitle" style="font-family: sans-serif; font-weight: bold;">Subtitle</h1>

    addClass

    addClass can be used to add classes to multiple elements at the same time. You should supply multiple classes in an array, and individual classes in a string. Since this function internally makes use of getElement, you can supply a range of elements to apply the class(es) to.

    Implementation:

    addClass: function(_class, element);

    Source:

    https://github.com/sharikul/F.js/blob/master/F.js#L455

    Returns:

    Array

    Predefined values:

    None

    Usage:

    Expose('addClass');

    <h1 id="title">Title</h1>
<p id="paragraph">Paragraph</p>

    .red {
 color: red;
}

    addClass('red', ['#title', '#paragraph']);

// OR
addClass(['red', 'blue'], ( ['#title', '#paragraph'] || '#title') );

    <h1 id="title" class="red blue">Title</h1>
<p id="paragraph" class="red blue">Paragraph</p>
